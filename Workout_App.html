Workout App

This is a workout tracking application where users can make their own workout plans. I made this web app in my free time to learn the NextJS framework as well as practice my learning of react. Some of the things I learned in this project include:

  • How to integrate databases (Supabase) for data storage and authentication for user profiles
  • Learned that the scale of Full-Stack applications is much larger than simpler projects. Requires a lot of planning about the site structure, functionality and how they affect each other
  • Intricacies related to NextJS Client and Server components and how they interact with React
  • Importance of finances on a project. The only free API I found for workout data was rate limited and didn't allow me to develop my app without having to worry about finances. Had to use an alternative hosted on JSONsilo
  • Setting realistic deadlines and avoiding burnout. I took 2 weeks more than anticipated to finish this project and by the end I did not spend as much time on the design as I wanted to. In future projects, I now know to do extensive planning before starting with coding
