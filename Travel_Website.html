Travel Website

This is a traveling website I created with the help of my groupmates in semester one. I did the programming for the website and these features are the highlights of what I learned how to do:

  • Interactive SVG map that reacts when you hover/click on it
  • Interactive map that has pins that can be inspected that give extra information
  • Filterable list of destinations
  • Google Street View implementation
